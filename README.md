# Personal Portfolio Website

My personal developer portfolio - designed and hand-coded from scratch, no templates, 
built to actually represent how I think about design and code, not just list my skills.

## Features

- 🎨 Custom lilac-themed design system with a hand-drawn underline accent and soft, 
  playful animations
- 🖱️ Interactive custom cursor with hover states
- 🎉 A small easter egg - try clicking my name
- 📱 Fully responsive, mobile-first layout with an animated mobile nav
- ✉️ Working contact form (Formspree) - sends real emails, no backend server
- ⬇️ Live CV download
- ♿ Built with accessibility in mind - keyboard navigation, reduced-motion support, 
  visible focus states

## Tech Stack

- **Languages:** HTML5, CSS3, JavaScript (vanilla, no frameworks)
- **Fonts:** Fraunces, Inter (Google Fonts)
- **Forms:** Formspree API
- **Hosting:** Netlify

## Why I built this

I wanted my portfolio to be a real project, not just a resume in website form, something 
that shows how I approach a UI from scratch: structure, animation, accessibility, and 
deployment, all without leaning on a template or a framework. It's also where I tied 
together my other projects in one place.
